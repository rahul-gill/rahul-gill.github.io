## Lists with create, delete, update operations
- there should be a search bar to search items
- FAB to add item
- long press reveeal a checkbox to the left of item, multiple items can be selected this way
- overflow menu or action in action bar to delete the items
- maybe an edit button on the right to edit name or another action in top bar which is only visible when single item is selected
- if item has many details click to go to detail screen and there have an edit button
- should have a confirmation dialog for deleting
