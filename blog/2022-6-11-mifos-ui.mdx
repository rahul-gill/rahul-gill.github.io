## Target Goals:
- Develop a common UI library to ensure a consistent look and feel of UIs for all apps
- Define a process for migrating apps to use the library
- Develop design guidelines and standards for the apps(design document)
- Develop base-level widgets, common UI components
- Implement screens and workflows to test out on reference sample app
- Document how to use the UI library and how to update the app when the library is updated.

## Challenge: Dependency management
- UI library will have the latest dependency versions. But apps' dependencies are not updated. Upgrading a single 
  implementation dependency version may require updating the compileSdkVersion, kotlinVersion, gradlePluginVersion 
  which may cause many problems.

## Themes and styles
- we'll have to extract common styles used in the apps and provide those styles
- apps consuming the styles would only apply these styles only(not provided by the material android libraries)
- text styles for these
  - link text for things like `forgot password` with click feedback(<u style={{"color":"#4444FF"}}>NonClicked</u> 
    <a style={{"color":"#AAAAFF"}}>Clicked</a>)
  - secondary text style for things like all rights reserved @copyright etc.,
  - list header style,
  - main focus text style(like the current account balance on home screen)
- Text Field Styles:
  - otp fill
  - password field
  - phone number select
  - edit text with action to fill the field(like qr code fetcher)
  - edit text with currency selection
- commonly used size and spacing values
  - text fields vertical padding and fillMaxWidth text fields horizontal padding
  - lists:
    - spacing between list items
    - card inner padding values
    - card outer padding values
    - item spacer horizontal padding values
    - bottom padding for avoiding list items hiding below the FAB
    - list item minHeight
    - list header left margin and vertical margins
  - images:
    - OrgIcon width and height
    - avatar width and height
    - card icon dimensions
    - basic icon dimension
  - title subtitle spacing
  - card corner radius and elevation
  - fab margins
  - standard margin
  - bottom navigation height and elevation
  - tooltip(hint) offset(x+,x-,y+,y-)
- text focus indication with alpha
